Установка Docker

Linux

curl -sSL https://get.docker.com/ | sh Установка Docker на Linux

Windows

https://download.docker.com/win/stable/InstallDocker.msi Скачать установщик для Windows

Mac

https://download.docker.com/mac/stable/Docker.dmg Скачать установщик для Mac

Реестры и репозитории Docker

Аутентификация

docker login Вход в реестр Docker Hub
docker login localhost:8080 Вход в локальный реестр
docker logout Выход из реестра

Поиск образов

docker search nginx Поиск образов nginx
docker search --filter stars=3 nginx Поиск образов с рейтингом от 3 звезд

Загрузка и выгрузка образов

docker image pull nginx Скачать образ nginx
docker image push username/nginx Загрузить образ в реестр

Управление контейнерами

Создание и запуск

docker container run -it ubuntu:latest Запуск контейнера в интерактивном режиме
docker container run -d -p 80:80 nginx Запуск в фоновом режиме с пробросом портов
docker container run -v /host:/container nginx Запуск с монтированием томов

Управление состоянием

docker container start mycontainer Запуск контейнера
docker container stop mycontainer Остановка контейнера
docker container restart mycontainer Перезапуск контейнера
docker container pause mycontainer Приостановка контейнера
docker container unpause mycontainer Возобновление контейнера

Просмотр информации

docker container ls Список запущенных контейнеров
docker container ls -a Список всех контейнеров
docker container logs mycontainer Просмотр логов контейнера
docker container inspect mycontainer Детальная информация о контейнере

Удаление

docker container rm mycontainer Удаление контейнера
docker container rm -f mycontainer Принудительное удаление контейнера

Управление образами

Просмотр и сборка

docker image ls Список образов
docker build -t myimage . Сборка образа из Dockerfile
docker image history myimage История сборки образа

Тегирование и экспорт

docker image tag nginx myrepo/nginx Тегирование образа
docker image save nginx > nginx.tar Экспорт образа в архив
docker image load < nginx.tar Импорт образа из архива

Удаление

docker image rm nginx Удаление образа
docker image prune Удаление неиспользуемых образов

Сеть в Docker

Управление сетями

docker network ls Список сетей
docker network create mynetwork Создание сети
docker network inspect mynetwork Информация о сети
docker network rm mynetwork Удаление сети

Подключение контейнеров

docker network connect mynetwork mycontainer Подключение контейнера к сети
docker network disconnect mynetwork mycontainer Отключение контейнера от сети

Очистка Docker

Очистка контейнеров

docker container prune Удаление остановленных контейнеров
docker rm $(docker ps -aq) Удаление всех контейнеров

Очистка образов

docker image prune Удаление неиспользуемых образов
docker image prune -a Удаление всех неиспользуемых образов

Полная очистка

docker system prune Очистка системы Docker
docker system prune -a Полная очистка системы

Docker Swarm

Инициализация и управление

docker swarm init Инициализация Swarm
docker swarm join-token worker Токен для подключения worker
docker swarm join-token manager Токен для подключения manager

Сервисы

docker service ls Список сервисов
docker service create --name web nginx Создание сервиса
docker service scale web=5 Масштабирование сервиса
docker service update --image nginx:latest web Обновление сервиса

Docker Compose

Основные команды

docker-compose up Запуск сервисов
docker-compose up -d Запуск в фоновом режиме
docker-compose down Остановка сервисов
docker-compose ps Статус сервисов
docker-compose logs Просмотр логов

Пример docker-compose.yml

version: '3.8'
services:
  web:
    image: nginx:alpine
    ports:
      - "80:80"
    volumes:
      - ./html:/usr/share/nginx/html
  db:
    image: postgres:13
    environment:
      POSTGRES_DB: myapp
      POSTGRES_USER: user
      POSTGRES_PASSWORD: password
    volumes:
      - db_data:/var/lib/postgresql/data

volumes:
  db_data:

Управление сервисами

docker-compose build Пересборка образов
docker-compose exec web bash Выполнение команды в сервисе
docker-compose restart Перезапуск сервисов
